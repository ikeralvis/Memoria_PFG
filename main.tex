\resumen{% 200-250 palabras
Este proyecto tiene como objetivo el diseño e implementación de un sistema inteligente integrado 
en una plataforma digital de comparación de precios de carburantes, desarrollada sobre Google 
Cloud Platform con una arquitectura de microservicios desplegada mediante Cloud Run y 
Cloud Functions. La solución, implementada con FastAPI en el backend y React en el frontend, 
permite a los usuarios consultar precios actualizados en tiempo real a partir de la API oficial 
del Ministerio para la Transición Ecológica, acceder a históricos de precios por estación 
de servicio y visualizar predicciones a siete días generadas mediante modelos de machine 
learning basados en LightGBM.
Más allá de la simple visualización de información, el sistema incorpora un módulo de 
recomendación de repostaje en ruta que, apoyándose en la API de Google Maps para el 
análisis geoespacial y la optimización de recorridos, determina qué estación resulta más 
conveniente económicamente para un trayecto definido por el usuario. Esta recomendación 
combina el precio actual, la evolución prevista y la desviación necesaria respecto al 
recorrido, evaluando el coste total del desplazamiento.
Los datos son gestionados mediante Neon Tech como almacén analítico principal y Cloud Storage 
para la persistencia de modelos y artefactos. La plataforma es accesible desde dispositivos 
móviles mediante una interfaz web responsive y contempla adicionalmente un asistente conversacional, priorizando en todo momento la 
experiencia de usuario.
El proyecto integra competencias de ingeniería de software, análisis de datos e inteligencia 
artificial aplicada, constituyendo una solución tecnológica orientada al ahorro económico, 
la eficiencia energética y la toma de decisiones basada en datos.}

\descriptores{Computación en la nube, predicción de precios, plataforma de carburantes, recomendación geoespacial.}

\chapter*{Definiciones, Acrónimos y Abreviaturas}

\section*{Acrónimos}
\label{sec:acronimos}

\begin{description}
  \item[API] \textit{Application Programming Interface}. Interfaz de programación de aplicaciones.
  \item[ARIA] \textit{Accessible Rich Internet Applications}. Conjunto de atributos para mejorar la accesibilidad de las aplicaciones web.
  \item[CDN] \textit{Content Delivery Network}. Red de distribución de contenido.
  \item[CI/CD] \textit{Continuous Integration / Continuous Deployment}. Integración continua y despliegue continuo.
  \item[CLS] \textit{Cumulative Layout Shift}. Cambio acumulado del diseño (métrica de estabilidad visual web).
  \item[CSP] \textit{Content Security Policy}. Política de seguridad de contenido.
  \item[DNS] \textit{Domain Name System}. Sistema de nombres de dominio.
  \item[DOM] \textit{Document Object Model}. Modelo de objetos del documento.
  \item[EV] \textit{Electric Vehicle}. Vehículo eléctrico.
  \item[FCP] \textit{First Contentful Paint}. Tiempo hasta el primer despliegue de contenido en pantalla.
  \item[GCP] \textit{Google Cloud Platform}. Plataforma de servicios en la nube de Google.
  \item[HSTS] \textit{HTTP Strict Transport Security}. Política de seguridad de transporte estricta para HTTP.
  \item[HTML] \textit{HyperText Markup Language}. Lenguaje de marcado de hipertexto.
  \item[HTTPS] \textit{HyperText Transfer Protocol Secure}. Protocolo seguro de transferencia de hipertexto.
  \item[JS] \textit{JavaScript}. Lenguaje de programación interpretado para entornos web.
  \item[JWT] \textit{JSON Web Token}. Estándar abierto para la transmisión segura de información mediante objetos JSON.
  \item[LCP] \textit{Largest Contentful Paint}. Tiempo hasta el despliegue del contenido más extenso en pantalla.
  \item[LLM] \textit{Large Language Model}. Modelo de lenguaje de gran escala.
  \item[MAE] \textit{Mean Absolute Error}. Error absoluto medio.
  \item[MAPE] \textit{Mean Absolute Percentage Error}. Error porcentual absoluto medio.
  \item[MCP] \textit{Model Context Protocol}. Protocolo abierto para la integración segura de modelos de lenguaje con fuentes de datos.
  \item[MITECO] Ministerio para la Transición Ecológica y el Reto Demográfico.
  \item[ML] \textit{Machine Learning}. Aprendizaje automático.
  \item[ODS] Objetivos de Desarrollo Sostenible (Organización de las Naciones Unidas).
  \item[ORS] \textit{OpenRouteService}. Servicio geográfico de enrutamiento basado en datos abiertos.
  \item[PFG] Proyecto Fin de Grado.
  \item[PostGIS] Extensión de objetos espaciales para la base de datos relacional PostgreSQL.
  \item[PWA] \textit{Progressive Web App}. Aplicación web progresiva.
  \item[RF] Requisito Funcional.
  \item[RGPD] Reglamento General de Protección de Datos.
  \item[SEO] \textit{Search Engine Optimization}. Optimización para motores de búsqueda.
  \item[STT] \textit{Speech-to-Text}. Tecnología de conversión de voz a texto.
  \item[TBT] \textit{Total Blocking Time}. Tiempo total de bloqueo del hilo principal.
  \item[TTS] \textit{Text-to-Speech}. Tecnología de síntesis de texto a voz.
  \item[WCAG] \textit{Web Content Accessibility Guidelines}. Pautas de accesibilidad para el contenido web.
  \item[XSS] \textit{Cross-Site Scripting}. Tipo de vulnerabilidad de inyección de código malicioso en aplicaciones web.
\end{description}
